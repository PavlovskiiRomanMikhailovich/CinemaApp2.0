# CinemaStrapi

Проект для поиска и просмотра информации о фильмах. Разработан на Next.js с использованием Strapi в качестве бэкенда.

### Установка и запуск

```bash
# Установка зависимостей
npm install
# или
yarn install

# Запуск в режиме разработки
npm run dev
# или
yarn dev
```

## Деплой

Приложение задеплоено на Vercel и доступно по ссылке:

https://cinema-app2-0.vercel.app/movies
